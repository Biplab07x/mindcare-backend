# MindCare Backend

This repository contains the backend server for the MindCare project.

## Purpose
The backend handles server-side logic for the MindCare platform, including request handling and chatbot logic (implemented using conditional if–else statements without external AI APIs).

## Tech Stack
- Node.js
- Express.js

## How to Run Locally
1. Navigate to the backend folder
2. Install dependencies:
   npm install
3. Start the server:
   node server.js

The server runs on:
http://localhost:5000

## Notes
- This backend does not use any third-party AI APIs
- The chatbot logic is rule-based (if–else)
- This project is built for a hackathon submission
